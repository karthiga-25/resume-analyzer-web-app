# 📄 Resume Analyzer Web App

This is a web application that analyzes resumes based on selected job roles and provides insights such as skill match, missing skills, and improvement suggestions.

---

## 📌 Description

The Resume Analyzer Web App allows users to paste resume text or upload a PDF file and select a target job role. The system analyzes the resume by matching it against predefined skill sets and provides a score along with missing skills and suggestions.

---

## 🛠️ Technologies Used

* 🐍 Python
* ⚡ FastAPI
* 🌐 HTML
* 🎨 CSS
* ⚡ JavaScript
* 📄 PDF Processing (pdfplumber)

---

## ✨ Features

* 📄 Analyze resume text
* 📤 Upload and analyze PDF resumes
* 🎯 Role-based skill matching
* 📊 Score calculation
* ❌ Missing skills identification
* 💡 Suggestions for improvement

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/karthiga-25/resume-analyzer-web-app.git
```

2. Navigate to the project folder:

```
cd resume-analyzer-web-app
```

3. Install dependencies:

```
pip install fastapi uvicorn pdfplumber
```

4. Run the backend server:

```
uvicorn main:app --reload
```

5. Open `index.html` in your browser
